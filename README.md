2023-09-01 ~ 2024-01-02

## 주요 기능
* 브랜드별 커피나 사이즈를 선택해 기록
* 목록에 없는 커피나 음료 타이핑으로 입력해서 기록
* 기록된 음료의 카페인수치를 나타내기
* 기록된 수치를 초기화
* 캘린더 날짜를 선택해서 해당 날짜에 기록된 카페인수치 확인
* 공지사항, 버전 정보, 이용방법, 팝업알림을 구현
